Weblinks Bookmark Manager

Save time and boost productivity with our simple and

effective web links management system!

Every Internet user sooner or later faces the problem: where to store bookmarks - links to frequently visited web resources? Bookmarks provided by browsers can partly solve this problem. But this functionality is limited and using browser bookmarks is inconvenient. Our site Bookmark Manager allows you to solve this problem once and for all.

After you register, you have the opportunity to create, store and use conveniently and clearly organized links to web resources. Create your own pages, rows within pages and columns within rows, then insert links within columns. Context menus like this

are used to add, change or delete them. To swap pages, rows, columns or links, just drag them with the mouse and drop.

Make https://weblinks.click the default page in your desktop browser or create a shortcut on your phone's home screen for quick access to your own links.

Google Chrome extension

If you are using Chrome browser, install weblinks.click for Chrome extension.

Having the extension installed go to the web page you are interested in, the link to which you would like to add to one of your pages on weblinks.click. Click on the extension button, then select the page/row/column where you want to add the link. Edit the url and label if necessary. Click the "Add link to weblinks.click" button. Go to the corresponding page in weblinks.click and refresh the page content. The link will be added.

Pages

At the top of the site there is a toolbar containing buttons with the titles of the pages you are creating. Clicking on a button will open the corresponding page. When no pages have been created yet, the toolbar will have an "Add page" button. Click this button to create your first page. In the form that appears, type in the path to the page, for example "services", and the page title, for example "My services" and click "Submit". The path is used in the URL as a unique address for the page on our site. The title will be shown on the toolbar button representing the page. To create a new page, edit or delete a page, right-click on the page button and use the context menu. To swap pages, drag the button with the mouse and drop it.

You can protect a page from accidental changes by marking it as "read-only".

In the toolbar, read only pages are displayed in italics.

Rows

A page contains rows that are used to place columns. If the page does not yet contain rows, the "Add row" button will be shown. Click on this button to create the first row. In the form that appears, type an optional row title. To create a new row, edit or delete a row, right-click on the row and use the context menu. To swap rows, drag the row with the mouse and drop.

Columns

The row contains columns that are used to place links. If the row does not yet contain columns, the "Add column" button will be displayed. Click on this button to create the first column. In the form that appears, type an optional column title. To create a new column, edit or delete a column, right-click on the column and use the context menu. To swap columns, drag the columns with the mouse and drop.

Bookmarks

The column contains your bookmarks - links to web resources. When you click on a link, the corresponding page opens in a new browser tab. If the column does not yet contain links, it will show the "Add link" button. Click on this button to create the first link. In the form that appears, in the "Url" field, type the Internet address of the page starting with the protocol, for example, https://google.com. To avoid typing the address manually, go to the page you are interested in in the browser, copy the address from the address bar, and then return back to the form and paste the address into the "Url" field. In the "Label" field, type a string that will represent the link in the column. To create a new, change or delete a link, right-click on the link and use the context menu. To swap links, drag the link with the mouse and drop it.
